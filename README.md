📈 Stock Market Price Prediction using Machine Learning

Overview

This project demonstrates how machine learning techniques can be used to analyze historical stock market data and predict stock price movements. The model uses financial data and technical indicators to identify patterns and estimate future price behavior.

Features

- Historical stock market data collection
- Technical indicators (Moving Average, RSI)
- Machine learning model implementation
- Stock price prediction
- Visualization of predicted vs actual prices
- Basic trading signal generation

Tools & Technologies

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Yahoo Finance API (yfinance)
